Hey there!


Do you want to make your very own quaver skin? Well, you can do it the traditional way - download the default skin and modify the files.
However, if you don’t have the time or the skills, this simple HTML will help you create your skin!


It works by prompting for uploads for each category (keys, sound effects, backgrounds…). If there’s a file missing it’ll default to the stock circles skin by the offical quaver team.


Then, you’ll be able to export it, and if you think it’s really good, you can upload it to the workshop so everyone can use it!


The code is really simple, so don’t expect crazy UI/UX or features.
